    password_required?() click to toggle source

    Override Devise to allow for Authentication or password.

    An invalid authentication is allowed for a new record since the record needs to first be saved before the authentication.user_id can be set.

    Calls superclass method 
    # File app/models/user.rb, line 39
def password_required?
  if authentications.empty?
    super || encrypted_password.blank?
  elsif new_record?
    false
  else
    super || encrypted_password.blank? && authentications.find{|a| a.valid?}.nil?
  end
end
    reverse_merge_attributes_from_auth(auth) click to toggle source

    Merge attributes from Authentication if User attribute is blank.

    If User has fields that do not match the Authentication field name, modify this method as needed.

    # File app/models/user.rb, line 53
def reverse_merge_attributes_from_auth(auth)
  auth.oauth_data.each do |k, v|
    self[k] = v if self.respond_to?("#{k}=") && self[k].blank?
  end
end
